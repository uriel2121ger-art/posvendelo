# Checklist Go-Live por sucursal

## Previo al despliegue

- [ ] Backup verificado de BD local/servidor
- [ ] Token/credenciales revisados (sin hardcode operativo)
- [ ] Rol de equipo definido (server vs client)
- [ ] Conectividad entre cajas y servidor validada
- [ ] Rollback técnico documentado y probado

## Durante despliegue

- [ ] Servicios levantan sin error
- [ ] Venta de prueba end-to-end completada
- [ ] Timbrado de prueba completado (si aplica)
- [ ] Sync de prueba exitoso
- [ ] Monitoreo/alertas sin eventos P1

## Posterior (24-72h)

- [ ] Sin incidentes críticos
- [ ] KPIs dentro de umbral
- [ ] Acta de aceptación de sucursal
- [ ] Lecciones aprendidas registradas
